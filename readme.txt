	学校的教务管理工作是学校的一项很重要的工作，是学校正常运作的关键环节。随着高校教育的不断发展，科技的不断进步，传统的教务管理工作已经不能适应新时代的需求，所以需要一个能有效管理学校师生情况，方便学生学习生活的教务管理平台。
	本系统开发了一个基于Java的教务管理系统，完成了对学生、教师等信息的管理功能，并且内置一个自动考试系统，增加了资料共享库，最后实现了不同用户的权限控制功能。本系统根据业务的需求分析，将用户划分为不同的角色，然后对不同角色进行功能分析和数据库设计，用MySQL数据库建了12张表来存储数据。系统采用基于Java的SSH框架来设计MVC模式，将系统与用户的交互页面放在表示层，用户的逻辑功能代码放在控制层，实现了系统表示层和业务层的分离。除此之外，系统还采用了AJAX技术，在验证考试时间和删除学生、教师等信息的时候可以不刷新页面，直接在后台处理业务并在页面展示结果信息。
	经过测试，系统设计符合要求，功能实现，运行稳定。
